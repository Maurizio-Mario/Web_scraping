"113" "		<h1 class=\"post-title\" id=\"post-1643\">rvest: easy web scraping with&nbsp;R</h1>"
"114" "		<p class=\"post-metadata\">November 24, 2014 in <a href=\"https://blog.rstudio.org/category/packages/\" rel=\"category tag\">Packages</a>		</p>"
"115" "	</div>"
"116" "	<div class=\"post-content\">"
"117" "		<p>rvest is new package that makes it easy to scrape (or harvest) data from html web pages, inspired by libraries like <a href=\"http://www.crummy.com/software/BeautifulSoup/\">beautiful soup</a>. It is designed to work with <a href=\"https://github.com/smbache/magrittr\">magrittr</a> so that you can express complex operations as elegant pipelines composed of simple, easily understood pieces. Install it with:</p>"
"118" "<pre class=\"r\"><code>install.packages(&quot;rvest&quot;)</code></pre>"
"119" "<div id=\"rvest-in-action\" class=\"section level2\">"
"120" "<h2>rvest in action</h2>"
"121" "<p>To see rvest in action, imagine we’d like to scrape some information about <a href=\"http://www.imdb.com/title/tt1490017/\">The Lego Movie</a> from IMDB. We start by downloading and parsing the file with <code>html()</code>:</p>"
"122" "<pre class=\"r\"><code>library(rvest)"
"123" "lego_movie &lt;- html(&quot;http://www.imdb.com/title/tt1490017/&quot;)</code></pre>"
"124" "<p>To extract the rating, we start with <a href=\"http://selectorgadget.com\">selectorgadget</a> to figure out which css selector matches the data we want: <code>strong span</code>. (If you haven’t heard of <a href=\"http://selectorgadget.com/\">selectorgadget</a>, make sure to read <code>vignette(&quot;selectorgadget&quot;)</code> &#8211; it’s the easiest way to determine which selector extracts the data that you’re interested in.) We use <code>html_node()</code> to find the first node that matches that selector, extract its contents with <code>html_text()</code>, and convert it to numeric with <code>as.numeric()</code>:</p>"
"125" "<pre class=\"r\"><code>lego_movie %&gt;% "
"126" "  html_node(&quot;strong span&quot;) %&gt;%"
"127" "  html_text() %&gt;%"
"128" "  as.numeric()"
"129" "#&gt; [1] 7.9</code></pre>"
"130" "<p>We use a similar process to extract the cast, using <code>html_nodes()</code> to find all nodes that match the selector:</p>"
"131" "<pre class=\"r\"><code>lego_movie %&gt;%"
"132" "  html_nodes(&quot;#titleCast .itemprop span&quot;) %&gt;%"
"133" "  html_text()"
"134" "#&gt;  [1] &quot;Will Arnett&quot;     &quot;Elizabeth Banks&quot; &quot;Craig Berry&quot;    "
"135" "#&gt;  [4] &quot;Alison Brie&quot;     &quot;David Burrows&quot;   &quot;Anthony Daniels&quot;"
"136" "#&gt;  [7] &quot;Charlie Day&quot;     &quot;Amanda Farinos&quot;  &quot;Keith Ferguson&quot; "
"137" "#&gt; [10] &quot;Will Ferrell&quot;    &quot;Will Forte&quot;      &quot;Dave Franco&quot;    "
"138" "#&gt; [13] &quot;Morgan Freeman&quot;  &quot;Todd Hansen&quot;     &quot;Jonah Hill&quot;</code></pre>"
"139" "<p>The titles and authors of recent message board postings are stored in a the third table on the page. We can use <code>html_node()</code> and <code>[[</code> to find it, then coerce it to a data frame with <code>html_table()</code>:</p>"
"140" "<pre class=\"r\"><code>lego_movie %&gt;%"
"141" "  html_nodes(&quot;table&quot;) %&gt;%"
"142" "  .[[3]] %&gt;%"
"143" "  html_table()"
"144" "#&gt;                                              X 1            NA"
"145" "#&gt; 1 this movie is very very deep and philosophical   mrdoctor524"
"146" "#&gt; 2 This got an 8.0 and Wizard of Oz got an 8.1...  marr-justinm"
"147" "#&gt; 3                         Discouraging Building?       Laestig"
"148" "#&gt; 4                              LEGO - the plural      neil-476"
"149" "#&gt; 5                                 Academy Awards   browncoatjw"
"150" "#&gt; 6                    what was the funniest part? actionjacksin</code></pre>"
"151" "</div>"
"152" "<div id=\"other-important-functions\" class=\"section level2\">"
"153" "<h2>Other important functions</h2>"
"154" "<ul>"
"155" "<li>If you prefer, you can use xpath selectors instead of css: <code>html_nodes(doc, xpath = &quot;//table//td&quot;)</code>).</p>"
"156" "</li>"
"157" "<li>"
"158" "<p>Extract the tag names with <code>html_tag()</code>, text with <code>html_text()</code>, a single attribute with <code>html_attr()</code> or all attributes with <code>html_attrs()</code>.</p>"
"159" "</li>"
"160" "<li>"
"161" "<p>Detect and repair text encoding problems with <code>guess_encoding()</code> and <code>repair_encoding()</code>.</p>"
"162" "</li>"
"163" "<li>"
"164" "<p>Navigate around a website as if you’re in a browser with <code>html_session()</code>, <code>jump_to()</code>, <code>follow_link()</code>, <code>back()</code>, and <code>forward()</code>. Extract, modify and submit forms with <code>html_form()</code>, <code>set_values()</code> and <code>submit_form()</code>. (This is still a work in progress, so I’d love your feedback.)</p>"
"165" "</li>"
"166" "</ul>"
"167" "<p>To see these functions in action, check out package demos with <code>demo(package = &quot;rvest&quot;)</code>."
